Frequently asked questions


TECHNICAL & LICENSING

What kind of equipment do I need when enrolling in this course?

You need at lease a PC or Laptop(Windows OS or Mac OS) At least 2GB RAM, 200GB Storage Space, Microphone, Webcam (Recommended but optional), Headset (Recommended but optional) & Internet Connection (Min 5 Mbps)

What kind of tools do I need during the training?

You need a text editor (Visual studio code) recommend, a Browser(Google Chrome) recommend but optional, Conferencing platform (zoom.us) Recommended but optional & Learning management system(Google classroom)

What can I build after finishing this course?

After completing the course, you will be able to build an entire Website and Web app from scratch using HTML5, Materialize (css framework) and pure vanilla JavaScript as well as work on medium to large-sized Web apps as part of a team.


DESIGN

Do I get a certificate of completion?

Yes, you get a signed certificate of completion once you finish the entire course.

Can I get a group discount?

Group discounts are available for groups of 5, 10, 15, 20 and above. Contact us on the DevLift Facebook page for more information.

How do you process my payment?

As of the moment we use Bank deposit to process your payments, soon we're going to use Stripe, Gcash or other money transfer transaction.

Description

This course introduces the learner to the basics of HTML and Cascading Style Sheets (CSS), up to advanced technology the most fundamental technologies in creating web pages & web app.

Even if you have zero programming experience, this course will take you from beginner to mastery. Learners will also be introduced to software tools professionals use to create professional looking web pages & web app. Modern websites consist of more than just HTML and CSS.

learners will also learn Materialize CSS (Frameworks) & pure vanilla JavaScript-- the industry standard to create dynamic pages that enhance the user experience. This course consists of comprehensive lectures and hands-on exercises designed to help learners create feature-rich web pages that make use of JavaScript.

This web development course is also a good introduction for people wanting to get started with computer programming. We'll take you step-by-step through live online tutorials and teach you everything you need to know to succeed as a web developer.

Throughout this course, we cover a massive amount of tools and technologies, including:
  • Front-End Web Development
  • HTML 5
  • CSS 3
  • Web Design
  • Materialize CSS (framework)
  • jQuery
  • Bash command line
  • Git, GitHub and Version Control
  • Javascript ES5-ES6-ES7
  • DOM Manipulation
  • Object oriented programming
  • REST API
  • Deployment with GitHub Pages, Netlify & more!
By the end of this course, you will be fluently programming and be ready to make any website you can dream of.
